✨ Features<br>
🧠 語音辨識（VOSK STT）<br>
☁️ MQTT 雲端 AI 通訊（HiveMQ）<br>
🔊 Google TTS 語音回覆<br>
⏰ 多組鬧鐘系統<br>
🌐 Web UI 遠端設定<br>
🎛️ 多種聊天模式（Chat Modes）<br>
🔋 省電模式 + Deep Sleep<br>

🔌 Hardware<br>
| ESP32-C3  | 主控        <br>
| INMP441   | I2S 麥克風 <br>
| MAX98357A | I2S 喇叭  <br>
| SSD1306   | OLED 顯示 <br>
| TTP223    | 觸摸按鍵    <br>
